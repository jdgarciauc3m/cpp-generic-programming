\subsection{Callables}

\begin{frame}[t,fragile]{What is a callable entity}
\begin{itemize}
  \item A callable is an entity that can be passed as an argument and invoked:
    \begin{itemize}
      \item \textmark{Function objects}: any type 
        for which the function call operator is defined.

      \item \textmark{Pointer to member}: includes member functions and 
            member objects.
    \end{itemize}

  \mode<presentation>{\vfill\pause}
  \item A number of entities in the standard library take callable objects
        to invoke (and possibly store them):
    \begin{itemize}
      \item \cppid{std::function} / 
            \cppid{std::move\_only\_function} (\textmark{C++23}).
      \item \cppid{std::bind} / 
            \cppid{std::bind\_front} /
            \cppid{std::bind\_back} (\textmark{C++23}).
      \item \cppid{std::reference\_wrapper}.
      \item \cppid{std::thread} / \cppid{std::jthread} / \cppid{std::async} / \cppid{call\_once} / \cppid{packaged\_task}.
    \end{itemize}
\end{itemize}
\end{frame}
